# Skill: 双足机器人电源管理

## 适用范围

双足人形机器人。

## 双足的电源特殊性

### 1. 高峰值电流需求

双足机器人在动态平衡控制中持续发出小幅高频电流脉冲。**峰值电流远高于平均电流**——
膝/髋关节在维持站立时也持续工作。

实际影响:
- 同等电池容量下,双足续航明显短于轮式或固定臂
- 突发动作(站起、跳、挥手)瞬时电流可达静态的 2-3 倍
- BMS 的过流保护更容易误触发

### 2. 摔倒断电风险

双足机器人摔倒时:
- 撞击可能导致电池接口接触不良
- 电池可能从接收槽脱出
- 关节冲击可能引起电流尖峰
- BMS 可能误判为短路而切断

诊断"突然断电"时,**先排除是否刚摔倒**。摔倒后断电属于设备保护,不是故障。

### 3. 长时间站立的隐性发热

双足机器人即使"什么都不做",静态站立也持续消耗能量:
- 多个关节维持微小力矩(姿态维持)
- 平衡控制器持续运行
- 这些电流被电池长时间提供,导致内阻发热

实际影响:**"站着没干啥但电池热"是双足正常现象**——和静止机器人不同。
诊断时不要因此误判为电池故障。

## 双足特定的诊断扩展

### 扩展 universal 的"电池-负载耦合"

双足要分四种状态:

| 状态 | 关节特征 | 电池特征预期 |
|---|---|---|
| 完全静止(passive 模式) | 零力矩 | 极低电流(< 1A) |
| 站立(stand 模式) | 微小持续力矩 | 低-中电流(2-5A,取决于姿态) |
| 行走 | 周期性峰值 | 中-高电流(5-15A 周期波动) |
| 跑/跳 | 高峰值 | 极高峰值(> 20A) |

如果电池显示电流和当前模式不匹配:
- 静止但电流高 → 关节卡死、控制异常或漏电
- 站立但电流极高 → 平衡控制器异常或姿态不当
- 行走但电流低 → 关节失能或负载脱落

### 扩展"热失控早期识别"

双足额外加一条:**异步关节高温 + 电池温升**。

如果某关节先升温(机械问题),持续运行可能拖累电池温度——
关节内阻增加 → 控制器为达到目标力矩输出更大电流 → 电池负担增加 → 温升。

诊断"电池温升"时,**先检查是否有异常热的关节**。这是双足特有的复合诊断。
